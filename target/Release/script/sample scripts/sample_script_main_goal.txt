[HEADER]
Title=Exemple de script - Objectif principal
Description=Ceci est un exemple de scénario pour le Capitalism Lab, listant toutes les variables sous la section [OBJECTIF PRINCIPAL].

//---------------------------------------------------------------------------------------------//

[MAIN GOAL]
Goal Title=Objectif principal
Goal Description=C'est votre objectif principal.

Goal Achieved Text=Félicitations ! Vous avez atteint votre premier objectif principal..
// Si celui-ci est défini, lorsque le joueur a atteint ce but, ce texte sera affiché sur l'écran des objectifs.

Goal Failed Text=Désolé, vous n'avez pas atteint votre objectif dans le délai imparti.
// Si celui-ci est défini, lorsque le joueur a échoué dans ce but, ce texte sera affiché sur l'écran des objectifs

Number of Game Years=20

//---------------------------------------------------------------------------------------------//
// Les valeurs cibles pour les objectifs principaux et/ou les sous-objectifs peuvent contenir l'une des lignes suivantes :

[MAIN GOAL VALUES]
Player Wealth=50

Player Company Annual Revenue=200
Player Company Annual Profit=20
Player Company Net Worth=80
Player Company Market Cap=120
//(unité de valeurs : million dollars)

Controlled Companies Annual Revenue=500
Controlled Companies Annual Profit=50
Controlled Companies Net Worth=160
Controlled Companies Market Cap=250

Number of Controlled Companies=2

Total number of Employees=95
Operating Profit Margin=10
Return on Equity=10
Stock Price=88
Ownership=60
Number of Product Classes Being Produced=3
Number of Product Types Being Produced=10

//---------------------------------------------------------------------------------------------//
// Vous pouvez inclure dans cette section l'un des objectifs suivants de domination du secteur :

[MAIN GOAL INDUSTRIES]
Retail
Fabrication
Farming
Raw Material Production
Real Estate
Media

//---------------------------------------------------------------------------------------------//
// Vous pouvez inclure dans cette section l'un des objectifs suivants de domination du marché des produits :
// Vous pouvez visualiser toutes les classes de produits dans le jeu en accédant au rapport de synthèse sur les produits et en consultant la liste des classes de produits en haut à gauche du rapport.

[MAIN GOAL PRODUCTS]
Apparel
Automobile
Beverage
Computer


//---------------------------------------------------------------------------------------------//
// Vous pouvez éventuellement définir des récompenses pour l'objectif principal et les objectifs secondaires. Un sous-objectif peut être fixé à
// impose des pénalités lorsque le joueur ne l'atteint pas dans le délai imparti.

[MAIN GOAL REWARDS]
Score change=50
Corporate Cash change=5000000
Personal Cash change=100000

// Vous pouvez modifier soit l'expertise basée sur les compétences de la personne, soit l'expertise basée sur le produit de la personne.
 Pour l'expertise basée sur les compétences, <le type d'expertise> peut être : vente au détail, fabrication, agriculture, recherche et développement, production de matières premières, marketing, formation
// Pour l'expertise basée sur les produits, <classe de produit> peut être l'une des classes de produits du jeu, comme l'habillement, l'automobile, les boissons, etc.

Player expertise change=Computer, 30
COO expertise change=Electronic Products, 50
CTO expertise change=research and development, 16
CMO expertise change=marketing, 18

Tech change=Notebook Computer, 30
// Tech change=<product type>, <change value -1000 to 1000>
// Cela modifiera le niveau technologique de votre entreprise pour le type de produit spécifié lorsque vous aurez atteint l'objectif.

Instant invention=Smart Phone
// Instant invention=<product type>
// Si cela est défini, le type de produit sera inventé instantanément par votre société lorsque vous aurez atteint l'objectif.
