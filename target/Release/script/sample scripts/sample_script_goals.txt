[HEADER]
Title=Exemple de script - Objectif principal et objectifs secondaires
Description=Ceci est un exemple de script pour Capitalism Lab, listant toutes les variables sous la section [OBJECTIF PRINCIPAL] et [OBJECTIF SECONDAIRE].

//-------------------------------------------------------------------//

[MAIN GOAL]
Goal Title=Objectif principal
Goal Description=C'est votre objectif principal.

Deadline=2010-06-30
// Vous pouvez fixer une date limite directe pour les objectifs ou préciser le nombre d'années de jeu qui est accordé au joueur pour atteindre l'objectif.
// Dans l'exemple actuel, nous avons fixé la date limite pour atteindre cet objectif.

[MAIN GOAL VALUES]
Player Company Annual Revenue=200

[MAIN GOAL REWARDS]
Score change=50

//-------------------------------------------------------------------//

[SUBGOAL]
Goal Title=Premier sous-objectif
Goal Description=C'est votre premier sous-objectif.

Goal Achieved Text=Félicitations ! Vous avez atteint votre premier sous-objectif.
Goal Failed Text=Désolé, vous avez raté votre premier sous-objectif.

// S'il s'agit d'une définition, l'image de l'objectif sera affichée sur le rapport d'objectif.
// L'image doit être au format JPG, stockée dans le sous-dossier¡°script\image¡± sous le dossier Capitalism Lab, et la taille de l'image ne doit pas être supérieure à 395x85.
Goal Image=subgoal image 1.jpg

// Ce sous-objectif ne sera pas disponible avant le 1er janvier 1991
Goal Available Date=1991-01-01

// L'échéance de ce sous-objectif est définie.
Deadline=1995-09-30

[SUBGOAL VALUES]
Player Company Annual Revenue=2

[SUBGOAL REWARDS]
Score change=10

// Si le joueur n'atteint pas ce sous-objectif avant la date limite, les sanctions suivantes lui seront imposées.
[SUBGOAL PENALTIES]
Score change=-5
Personal Cash change=-1000000
COO expertise change=farming, -15
Tech Change=Jeans, -50

//-------------------------------------------------------------------//

[SUBGOAL]
Goal Title=Second sous-objectif
Goal Description=C'est votre second sous-objectif.

Goal Achieved Text=Félicitations ! Vous avez atteint votre deuxième sous-objectif.
Goal Failed Text=Désolé, vous avez raté votre deuxième sous-objectif.

// S'il est défini, cet objectif ne sera disponible que lorsque le joueur aura atteint avec succès un autre objectif que vous avez spécifié (objectif parent).
// Dans l'exemple actuel, le joueur doit d'abord atteindre le "premier sous-objectif" avant que le "deuxième sous-objectif" ne soit disponible.
Parent Goal=Premier sous-objectif

Number of Game Years=8

[SUBGOAL VALUES]
Ownership=80

[SUBGOAL REWARDS]
Score change=20

[SUBGOAL PENALTIES]
Score change=-10
